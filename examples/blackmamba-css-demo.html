CSS Variable System

Color Variables

:root { --primary-color: #007bff; --secondary-color: #6c757d; --success-color: #28a745; --danger-color: #dc3545; }

Spacing Variables

:root { --spacing-xs: 0.25rem; --spacing-sm: 0.5rem; --spacing-md: 1rem; --spacing-lg: 1.5rem; }

Light/Dark Mode Support

This demo automatically switches between light and dark mode based on your system preferences.

@media (prefers-color-scheme: dark) { :root { --primary-color: #0056b3; --text-color: #f0f0f0; --bg-color: #1a1a1a; } }

Key Benefits

  • Consistency: All components use the same variable system
  • Maintainability: Change themes by updating variables in one place
  • Performance: Single CSS file reduces HTTP requests
  • Accessibility: Built-in focus states and contrast ratios
  • Responsive: Mobile-first design with CSS variable adjustments
  • Collaboration: Clear separation between HTML (HTMX) and CSS (Web Designer)

This demo shows how BlackMamba's Web Designer Agent would create CSS components that seamlessly integrate with HTMX fragments created by the HTMX Agent.
